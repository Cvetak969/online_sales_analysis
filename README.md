# Online Sales Analysis

## Opis
Projekat za upravljanje proizvodima i analizom prodaje.

## Klase

### Product
- name, price, quantity
- prikaz i ažuriranje

### ProductManager
- dodavanje, prikaz, ukupna vrednost
- uklanjanje proizvoda

### Cart
- dodavanje u korpu
- ukupna cena
- prikaz korpe

## Funkcionalnosti
- upravljanje inventarom
- simulacija kupovine
- Git verzionisanje sa granama
